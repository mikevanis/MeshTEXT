MeshText Firmware

Peer-to-peer teletext mesh on ESP32 + LoRa

Connect your Heltec LoRa 32 V3 via USB and click Update.

Your browser does not support Web Serial.
Use Chrome or Edge on desktop. Serial access not allowed.
Make sure the page is served over HTTPS.

Your pages and settings are preserved.

First time? Full install includes bootloader and partition table.

Getting Started

What You Need

A Heltec LoRa 32 V3 (or V3.1) board with a LoRa antenna connected.

Controls

There's one button (the USER button, next to USB):

Click
Scroll through items
Double-click
Select / open
Hold
Go back
Hold 3 seconds
Enter / exit edit mode

Browsing

  • On boot you'll see a directory of pages
  • Click to scroll, double-click to open a page
  • Nearby MeshText nodes appear automatically
  • Select a remote node to browse their pages

Creating Pages

  • Hold the button for 3 seconds to start WiFi edit mode
  • Connect your phone to the "MeshTxt-XXXX" WiFi network
  • A page editor opens in your browser
  • Create pages with text and block graphics
  • Hold 3 seconds again or reset board to exit edit mode

How it Works

Every node broadcasts its presence over LoRa radio. When you view another node's page, your device requests it over the mesh. Nodes in between automatically relay messages for each other.

Requires Chrome or Edge with Web Serial API support.
